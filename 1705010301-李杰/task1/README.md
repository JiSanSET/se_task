# Excecise1

软件工程作业

##使用说明

根据提示输入年份，选则计算机字长，输入程序员员每天开发的指令数，输入程序员的月平均工资
即可计算出相应的存储容量需求，存储器价格，以及成本

开发工具：andriod studio
运行环境：安卓手机或安卓虚拟机（android 版本不低于4.0）

##文件

    MyApplication.zip包含了所有的项目文件
    app-release.apk为生成的apk,安装即可使用
